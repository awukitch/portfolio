Core4 Tools in Python!

Loops

Definition:

a set of instructions that restarts from the beginning either infinitly or until a certain condition is met

Example:

Real Life:

A loop is like going down the slide and once you reach the bottom you climb the stairs to return to the top again

Functions

Definition:

a set of instructions that are grouped together

Example:

Real Life:

A function could be used to make a peanutbutter and jelly sandwhich; first get out the bread, jelly, and peanutbutter, next spread the peanutbutter on one side of a piece of bread, next spread jelly on the other piece of bread, finally put the two slices together and enjoy!

Conditionals

Definition:

a statement that changes the output based on the conditions met

Example:

Real Life:

A conditional can be used to decided if you need to stop for gas: if the tank is almost out, stop for gas, if the tank is full do not get gas

Variables

Definition:

an element that can be changed

Example:

Real Life:

A variable is like the outfit you wear each day, you can change it day to day
